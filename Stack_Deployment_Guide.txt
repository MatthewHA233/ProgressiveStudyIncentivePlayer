# 华为交换机堆叠部署指南

## 拓扑概述
本配置实现了两个堆叠组的网络拓扑：

### 堆叠组1 (Domain 1)
- LSW1: 主设备，优先级200
- LSW2: 备用设备，优先级150

### 堆叠组2 (Domain 2)  
- LSW4: 主设备，优先级200
- LSW3: 备用设备，优先级150
- LSW5: 最低优先级设备，优先级100

## 部署步骤

### 1. 物理连接
按照拓扑图进行物理线缆连接：
- 堆叠组1内部：LSW1和LSW2通过千兆接口GE0/0/23-24连接
- 堆叠组2内部：LSW3、LSW4、LSW5通过千兆和以太网接口连接
- 组间连接：两个堆叠组通过多条链路互连

### 2. 配置顺序
建议按以下顺序配置设备：
1. 先配置各堆叠组的主设备（LSW1、LSW4）
2. 再配置备用设备（LSW2、LSW3）
3. 最后配置LSW5

### 3. 验证命令
配置完成后使用以下命令验证：

```
# 查看堆叠状态
display stack

# 查看堆叠成员
display stack member

# 查看堆叠拓扑
display stack topology

# 查看接口状态
display interface brief
```

### 4. 故障排除
常见问题及解决方法：
- 堆叠链路不通：检查物理连接和接口配置
- 主备切换异常：检查优先级配置和域设置
- 设备无法加入堆叠：检查软件版本兼容性

## 注意事项
1. 所有设备需要运行相同的软件版本
2. 堆叠链路应使用专用堆叠端口
3. 建议先在测试环境验证配置
4. 部署前做好配置备份
5. 确保电源冗余配置
6. 堆叠系统内置防环机制，无需额外配置STP

## 配置特点
- 纯堆叠配置，专注于堆叠功能
- 去除了不必要的VLAN和STP配置
- 简洁高效，便于部署和维护
